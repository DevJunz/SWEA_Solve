# 입력 체크

# 문제 풀이

현재 숫자를 string으로 변환한다.<br>
그 후 각 문자열의 문자를 추출하여 3,6,9 여부를 확인한 뒤<br>
빈 String clap에 "-"를 추가하는 방식을 사용했다.
